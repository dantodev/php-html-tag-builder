# PHP HTML tag builder
HTML tag builder for PHP.

#### Dependencies

* `PHP >= 5.6.0`


#### Installation

Install with [Composer](http://getcomposer.org):
```
composer require dtkahl/php-html-tag-builder
```

### Usage

```
new HtmlTagBuilder("div", ['class' => 'foo']);
```
